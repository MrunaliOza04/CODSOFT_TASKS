# CODSOFT Java Development Internship

This repository contains all the tasks completed during my Java Development Internship at CODSOFT.

## Tasks Completed

### Task 1 - Number Guessing Game
A console-based game where the user guesses a randomly generated number.

### Task 2 - Student Grade Calculator
Calculates total marks, average percentage, and grade based on user input.

### Task 3 - ATM Interface
A menu-driven ATM application supporting balance inquiry, deposit, and withdrawal operations.

### Task 4 - Quiz Application
A console-based quiz system that allows users to answer multiple-choice questions and displays their final score.

### Task 5 - Course Registration System
A course registration application where students can view available courses, register, drop courses, and view their registered courses.
