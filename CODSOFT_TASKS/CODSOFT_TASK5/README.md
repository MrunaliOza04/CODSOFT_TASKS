# CODSOFT Java Development Internship

## Task 5 - Course Registration System

### Description
This is a console-based Java application that simulates a course registration system for students. It allows users to view available courses, register for courses, drop registered courses, and view their registered courses through a menu-driven interface.

### Features
- View all available courses
- Register for a course
- Drop a registered course
- View registered courses
- Prevents duplicate registrations
- Checks course seat availability
- Handles invalid inputs gracefully

### Technologies Used
- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- ArrayList Collections
- Scanner Class

### How to Run

1. Open the project in IntelliJ IDEA.
2. Run `Main.java`.
3. Select an option from the menu.
4. Enter the required student and course details when prompted.

### Functionalities

| Option | Description |
|---------|-------------|
| View Available Courses | Displays all available courses with details |
| Register for Course | Registers a student for a selected course |
| Drop Course | Removes a registered course for a student |
| View Registered Courses | Displays all courses registered by a student |
| Exit | Closes the application |
