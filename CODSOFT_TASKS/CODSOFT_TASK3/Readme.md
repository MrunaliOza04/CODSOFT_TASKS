# CODSOFT Java Development Internship

## Task 3 - ATM Interface

### Description

A console-based ATM Interface developed in Java. The application allows users to perform basic banking operations such as checking account balance, depositing money, and withdrawing money.

### Features

- Check account balance
- Deposit money
- Withdraw money
- Input validation
- Insufficient balance validation
- Menu-driven interface

### Technologies Used

- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- Scanner Class

### Project Structure

```
ATM.java
BankAccount.java
Main.java
```

### How to Run

1. Open the project in IntelliJ IDEA.
2. Run `Main.java`.
3. Choose the desired ATM operation.
