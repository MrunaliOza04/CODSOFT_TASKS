# CODSOFT Java Development Internship

## Task 2 - Student Grade Calculator

### Description
This is a console-based Java application that calculates a student's total marks, average percentage, and grade based on the marks entered for each subject.

### Features
- Input marks for multiple subjects
- Calculates total marks
- Calculates average percentage
- Assigns grade based on percentage
- Validates marks (0–100)
- Displays final result

### Technologies Used
- Java
- IntelliJ IDEA
- Scanner Class

### How to Run

1. Open the project in IntelliJ IDEA.
2. Run `StudentGradeCalculator.java`.
3. Enter the number of subjects and marks when prompted.

### Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90–100 | A+ |
| 80–89 | A |
| 70–79 | B |
| 60–69 | C |
| 50–59 | D |
| Below 50 | F |
