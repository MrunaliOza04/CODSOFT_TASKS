# CODSOFT Java Development Internship

## Task 1 - Number Guessing Game

### Description
This is a console-based Java application where the computer generates a random number between 1 and 100. The player has to guess the number within a limited number of attempts.

### Features
- Generate a random number between 1 and 100
- User enters guesses
- Feedback for High, Low, or Correct guess
- Maximum 10 attempts
- Multiple rounds
- Score calculation

### Technologies Used
- Java
- IntelliJ IDEA
- Scanner Class
- Random Class

### How to Run

1. Open the project in IntelliJ IDEA.
2. Run `NumberGuessingGame.java`.
3. Follow the on-screen instructions.
